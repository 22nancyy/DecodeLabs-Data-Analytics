# DecodeLabs-Data-Analytics
A collection of Data Analytics projects completed during the DecodeLabs Data Analytics Internship, covering data cleaning, data preprocessing, exploratory data analysis (EDA), visualization, and business insights using Python and Pandas.
📊 Project 01 – Data Cleaning & Preprocessing
🎯 Objective

Prepare a raw e-commerce sales dataset for analysis by improving data quality, ensuring consistency, and removing inaccuracies through data cleaning and preprocessing techniques.

📌 Tasks Performed
Imported and explored the dataset.
Checked dataset structure, data types, and missing values.
Removed duplicate records and validated unique Order IDs.
Standardized text formatting for categorical columns.
Verified numerical values and ensured data consistency.
Prepared an analysis-ready dataset for downstream tasks.
📊 Key Observations
The dataset contained 1,200 records with 14 features.
No major missing values remained after preprocessing.
Duplicate and inconsistent records were successfully handled.
Data types were validated for accurate analysis.
💡 Business Insights
Clean and consistent data improves the reliability of business decisions.
Data validation reduces reporting errors and enhances analytical accuracy.
A well-prepared dataset forms the foundation for meaningful exploratory analysis.

📈 Project 02 – Exploratory Data Analysis (EDA)
🎯 Objective

Analyze the cleaned sales dataset to uncover trends, customer behavior, and revenue patterns using descriptive statistics and visual exploration.

📌 Tasks Performed
Performed descriptive statistical analysis.
Calculated mean, median, and total revenue.
Analyzed product-wise sales and payment methods.
Identified referral source performance.
Performed correlation analysis and outlier detection.
Generated insights using visualizations.
📊 Key Observations
Generated business metrics such as total revenue and average order value.
Compared product performance and payment preferences.
Identified relationships among numerical variables.
Detected outliers in transaction values.
💡 Business Insights
Product demand varies significantly across categories.
Customer payment preferences influence purchasing behavior.
Marketing channels contribute differently to customer acquisition.
Revenue is primarily influenced by quantity and unit price.
